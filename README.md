# Remote Process Monitor

Python application for monitoring running applications on a computer.

## Features

- monitors configured processes using **psutil**
- reads tracked applications from **config.json**
- logs monitoring results to a log file
- graphical interface built with **tkinter**
- background monitoring using **threading**
- real-time log updates through **queue**

## Technologies

- Python
- psutil
- tkinter
- threading
- JSON
- logging
